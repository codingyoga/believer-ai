## Installation

1. Clone the repo:
    ```sh
    git clone https://github.com/codingyoga/believer-ai.git
    ```

2. Navigate to the project directory:
    ```sh
    cd believer-ai
    ```

3. Install dependencies:
    ```sh
    pip install -r requirements.txt
    ```

4. Run the app:
    ```sh
    streamlit run believer-ai.py
    ```
